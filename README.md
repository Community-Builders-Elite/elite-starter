# Your Coach Starter Kit

You're in. This folder is your office for working with Claude Code, a smart AI helper that helps you run your coaching business.

You do not need to be techy. If you can type and follow simple steps, you can do this.

---

## What's in here?

- **CLAUDE.md**, your "business brain." It tells Claude about you so it can sound like you.
- A few **folders** to keep your work organized (reference, research, outputs).
- Some **skills**, which are shortcuts that do jobs for you.

You don't have to understand any of that yet. The guide explains it all.

---

## Start here (literally)

Make sure Claude Code is running in this folder, then type this and press Enter:

```
/start-here
```

That kicks off a friendly guide made just for you. It teaches you everything one small step at a time, and you can stop after any lesson and pick up later.

Prefer to go slow? You can do one small task a day for seven days. Just tell Claude "let's do the 7 day plan."

---

## You can't break anything

Really. Click around, try things, ask questions. If you ever get stuck, just type your question to Claude in plain words, like "what does this do?" or "I'm confused, help me."

You always have a helper right next to you. Type `/start-here` and let's go.
