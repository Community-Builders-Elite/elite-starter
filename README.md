# Welcome to Your Coach Starter Kit

This is your starting point for using Claude Code, a smart AI helper that works on your computer and helps you run your coaching business.

You do not need to be techy. You do not need to know how to code. If you can type and follow simple steps, you can do this.

There are a few things to set up first. Do them in order. It takes about 20 minutes, and you only do it once.

---

## What is this?

Think of this folder as your office. It comes with:

- A file called `CLAUDE.md`. This is your "business brain." It tells Claude about you so it can sound like you.
- A few folders to keep your stuff organized.
- Some helpers (called "skills") that do jobs for you.

When you open this folder, Claude reads all of it, so it always knows your business.

---

## Get set up (do these in order)

### 1. Get a free GitHub account

GitHub is a free website that keeps a safe backup of your work online. Think of it like Google Drive, but for projects like this one.

Go to **github.com** and sign up. It's free. Remember your username and password, you'll use them in a minute.

### 2. Install VS Code

VS Code is the free program where you and Claude work together. Think of it as your desk.

Go to **code.visualstudio.com**, download it, and install it. Just click through the setup.

### 3. Install Claude Code

Claude Code is the AI helper itself.

Go to **docs.claude.com/claude-code** and follow the install steps for your computer. You only install it once.

### 4. Get your own copy of this kit

Right now you're looking at the starter kit. You want your OWN copy to work in, so you don't change the original.

The easiest way: open VS Code, start Claude Code (see the next section), and just say:

> "Help me make my own copy of this starter kit and connect it to my GitHub."

Claude will walk you through it, step by step. You don't have to figure it out alone.

---

## How to start Claude Code

1. Open VS Code.
2. Click **Terminal** at the top, then **New Terminal**. A small panel opens at the bottom. (The terminal is just a box where you type one command. Don't worry about it.)
3. In that panel, type this and press Enter:

   ```
   claude
   ```

4. Claude starts up. Now you can talk to it in plain words, like texting a smart friend.

---

## What to do first

Once Claude is running, type this and press Enter:

```
/start-here
```

This starts a friendly guide made just for you. It teaches you everything, one small step at a time:

1. Making this your own workspace
2. Filling in your business brain
3. Saving and backing up your work
4. How Claude remembers things
5. Skills (your shortcuts)
6. Connecting your other tools
7. Keeping your costs low

You can do one lesson and stop. When you come back, type `/start-here` again and it picks up where you left off.

---

## You can't break anything

Really. Click around, ask questions, try stuff. If you ever get confused, just type your question to Claude in plain words, like "what does this do?" or "I'm stuck, help me."

You always have a helper right next to you. That's the whole point.

Type `/start-here` and let's go.
