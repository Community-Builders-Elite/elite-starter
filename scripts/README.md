# scripts

This folder holds small helper tools (little programs) that Claude can run for you.

## What goes here

- Simple automations Claude builds for you over time
- Helper tools that do a repeatable job, like pulling a report or formatting a file

## Do I need to touch this?

No. Most coaches never open this folder by hand. When you ask Claude to automate something, it may save a tool here and run it for you. It's just a tidy home for that kind of thing.

If you're brand new, you can ignore this folder entirely. It's here so your workspace is ready when you grow into it.
