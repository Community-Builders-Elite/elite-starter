# First 7 Days, Classroom Lessons

These are the written daily lessons for the "First 7 Days in Claude Code" classroom folder.

Each `day-N.md` is one full lesson, ready to paste into the matching Skool lesson. The coach reads the lesson in the classroom, then opens Claude Code and says "let's do Day N" to do the hands-on action, which /start-here walks them through.

One file per day:
- `day-1.md` Make It Yours
- `day-2.md` Tell Claude About Your Business
- `day-3.md` Save Your Work the Safe Way
- `day-4.md` Get Organized (and Do It for Real)
- `day-5.md` Build Your Own Skill
- `day-6.md` Connect a Tool You Use
- `day-7.md` Keep Your Costs Low (and Practice It)

The matching in-repo guide is `/start-here`. The one-page overview version is `.claude/skills/start-here/references/first-7-days.md`.
