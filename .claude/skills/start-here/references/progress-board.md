# Progress Board Template

Print this every run. Fill the boxes from `.start-here-state.json`:
- `[x]` if the lesson number is in `lessons_done`
- `[>]` for the current lesson the coach is on
- `[ ]` for lessons not started yet

Keep it exactly this simple. Do not add columns or extra detail.

```
LEARNING CLAUDE CODE, your progress

[ ] Lesson 1: Make This Your Own
[ ] Lesson 2: Your CLAUDE.md
[ ] Lesson 3: Save Your Work
[ ] Lesson 4: Memory and Your Folders
[ ] Lesson 5: Skills
[ ] Lesson 6: Connecting Tools
[ ] Lesson 7: Cost and Savings Tips

YOU ARE ON: Lesson N, <lesson name>

NEXT: <one short sentence about what you'll do in this lesson>
```

After the board, go straight into teaching the current lesson. Do not list all the lessons again or give them a menu. One next step only.
